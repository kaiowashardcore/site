# Kaiowas HC Website

Site experimental desenvolvido para a Kaiowás HC.
O intuito é ser um ecossistema para a banda, onde quem nos acompanha
pode ter acesso a agenda de shows, musicas, letras e tambem
a uma loja.

# Funcionalidades

Loja com funcionalidade de calculo de frete, sistema de carrinho
mensagens diretas para finalidades de compra e por fim um sistema de notificacoes para lancamentos de musicas e novos produtos de merchandising
